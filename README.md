# 苏大双创平台刷课系统

智能自动化刷课工具，支持苏州大学双创平台课程自动学习。

## 功能特点

- 🎯 **智能时长检测** - 自动识别视频实际时长
- 🛡️ **多重验证机制** - 确保学习记录有效提交
- 🚫 **防暂停保护** - 防止视频意外暂停
- 🪟 **弹窗处理** - 自动检测并处理选择题弹窗
- ⚡ **快速完成** - 支持快速完成模式
- 📊 **实时进度** - 可视化学习进度展示

## 技术特性

- 基于Tampermonkey用户脚本
- 智能API调用监控
- 多重完成验证
- 状态持久化存储

## 安装方法

### 前置要求
1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)
2. 确保可以访问苏州大学双创平台

### 安装步骤
1. 点击本项目的 `.user.js` 文件
2. 点击"Raw"获取原始文件
3. Tampermonkey会自动检测并提示安装
4. 确认安装后即可使用

## 使用方法

### 课程列表页面
- 点击"自动开始第一个课程"按钮
- 系统会自动跳转到第一个课程

### 视频播放页面
- **智能刷课**：模拟真实学习进度，推荐使用
- **快速完成**：立即完成当前课程
- **停止**：停止当前刷课进程
